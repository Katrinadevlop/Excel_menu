# Создание и установка EXE версии приложения

## Создание исполняемого файла (.exe)

### Шаг 1: Запуск сборки
1. Запустите файл `build_exe.bat` двойным кликом
2. Дождитесь завершения процесса сборки
3. После успешной сборки в папке `dist` появится файл `MenuApp.exe`

### Что делает build_exe.bat:
- Активирует виртуальную среду Python (если есть)
- Устанавливает PyInstaller (если не установлен)
- Удаляет старые файлы сборки
- Создает исполняемый файл с помощью PyInstaller
- Упаковывает все необходимые файлы (включая templates)

## Установка на другом компьютере

### Подготовка к установке:
1. Скопируйте всю папку `dist` на другой компьютер
2. В папке dist должны быть:
   - `MenuApp.exe` - основное приложение
   - папка `templates` - шаблоны Excel и PowerPoint
   - другие необходимые файлы

### Способ 1: Автоматическая установка
1. Поместите файл `install.bat` в папку `dist`
2. Запустите `install.bat` с правами администратора
3. Программа установится и создаст ярлык на рабочем столе

### Способ 2: Ручная установка
1. Создайте папку для приложения (например, `C:\MenuApp`)
2. Скопируйте все содержимое папки `dist` в созданную папку
3. Создайте ярлык на `MenuApp.exe` на рабочем столе

## Системные требования

### На компьютере разработчика (для сборки):
- Windows 10/11
- Python 3.8+
- Все зависимости из requirements.txt

### На целевом компьютере (для установки):
- Windows 10/11
- Не требуется Python или дополнительные библиотеки
- Около 200-300 МБ свободного места

## Структура файлов для распространения

```
dist/
├── MenuApp.exe          # Основное приложение
├── templates/           # Шаблоны
│   ├── presentation_template.pptx
│   ├── Бракеражный журнал шаблон.xlsx
│   └── Шаблон меню пример.xlsx
└── install.bat         # Скрипт установки (опционально)
```

## Устранение неполадок

### Если сборка не удалась:
1. Убедитесь, что установлены все зависимости: `pip install -r requirements.txt`
2. Убедитесь, что PyInstaller установлен: `pip install pyinstaller`
3. Проверьте, что все пути в spec-файле корректны

### Если приложение не запускается на другом ПК:
1. Убедитесь, что скопированы все файлы из папки `dist`
2. Проверьте, что папка `templates` находится рядом с `MenuApp.exe`
3. Запустите приложение от имени администратора
4. Проверьте совместимость версий Windows

### Если файлы Excel не обрабатываются:
1. Убедитесь, что на целевом ПК установлен Microsoft Excel или LibreOffice
2. Проверьте права доступа к файлам Excel
3. Убедитесь, что файлы Excel не повреждены

## Размер и производительность

- Размер exe файла: ~150-200 МБ (включает все Python библиотеки)
- Время запуска: 3-5 секунд (первый запуск может быть дольше)
- Требования к памяти: ~50-100 МБ ОЗУ

## Обновления

Для создания новой версии:
1. Внесите изменения в код
2. Запустите `build_exe.bat` снова
3. Распространите новую папку `dist`
